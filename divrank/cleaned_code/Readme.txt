Before using this matlab package, please install the package by running install.m in matlab. When you want to uninstall the code, just run uninstall.m in matlab. 

Directories:
/core_code/: this directory includes the matlab implement of DivRank, Cumulative DivRank, (personalized) PageRank and MMR. 
/demo/: this directory includes the demo codes to run the three real examples in the paper. In all demo code files, all parameters are set to their optimal values. 
/data/: this directory includes all data sets used in this paper. 


****************************************************************
